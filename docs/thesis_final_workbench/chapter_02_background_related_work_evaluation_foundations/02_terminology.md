# Chapter 2: Background, Related Work, and Evaluation Foundations - Terminology

## dialogue state

- Definition: structured representation of known goals, constraints, proposed route, and progress

## construct validity

- Definition: degree to which a metric measures the intended construct

## lexical metric

- Definition: surface text-overlap metric; useful context but not route validation

## semantic ASR metric

- Definition: speech-understanding metric focused on preserving task meaning and entities

## error propagation

- Definition: downstream effects of an earlier pipeline error

## Use Rule

- Define a term when it first becomes necessary for the argument.
- Do not introduce terms only because they appear in the software.
