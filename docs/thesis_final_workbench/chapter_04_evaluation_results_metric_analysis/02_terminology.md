# Chapter 4: Evaluation, Results, and Metric Analysis - Terminology

## task success rate

- Definition: successful completed runs divided by completed runs

## semi-success

- Definition: valid or partial task progress without full constraint/task satisfaction

## audio-text delta

- Definition: audio success rate minus paired text success rate

## route optimality gap

- Definition: proposed route duration minus constraint-layer optimal route duration

## metric role

- Definition: outcome-confirming, diagnostic phase, associated diagnostic, supplementary, metric quality, or execution/constant

## Use Rule

- Define a term when it first becomes necessary for the argument.
- Do not introduce terms only because they appear in the software.
