# Chapter 1: Introduction - Terminology

## spoken dialogue system

- Definition: dialogue system using spoken input and/or spoken output

## task-oriented dialogue

- Definition: dialogue directed toward an externally checkable goal

## phase-wise evaluation

- Definition: evaluation using intermediate pipeline evidence instead of only final outcome

## task-grounded evaluation

- Definition: evaluation based on an external task environment such as the route network

## Use Rule

- Define a term when it first becomes necessary for the argument.
- Do not introduce terms only because they appear in the software.
