# Chapter 3: Methodology and Experiment Design - Terminology

## experimental unit

- Definition: one condition/run with one selected configuration

## knowledge boundary

- Definition: rule defining what each agent can know directly

## paired run

- Definition: text and audio run sharing the same non-audio configuration

## retrospective metric calculation

- Definition: metrics calculated after execution from preserved evidence

## constraint layer

- Definition: task stage where additional user constraints are revealed and optimal route changes

## Use Rule

- Define a term when it first becomes necessary for the argument.
- Do not introduce terms only because they appear in the software.
