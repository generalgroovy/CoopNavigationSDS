# Chapter 5: Discussion, Conclusion, and Future Work - Outline

Use this as the direct writing plan for the final compact thesis structure.

## Function

- Answer research questions directly.
- State contributions and validity boundaries.
- Keep future work short and tied to limitations.

## Sections

### 5.1 Research Question Answers

- phase-wise evidence is feasible
- task-grounded and semantic metrics are strongest
- speech channel reduces success in matched comparisons
- backend ranking remains cautious

### 5.2 Interpretation

- success alone is too coarse
- semi-success is analytically valuable
- failure localization identifies likely origin but not definitive cause

### 5.3 Validity Boundaries

- simulated caller
- synthetic network
- selected TTS/ASR
- descriptive correlations
- matched-subset limits

### 5.4 Contribution

- framework
- logging design
- metric validity classification
- matched text/audio analysis
- result interpretation method

### 5.5 Future Work

- human validation
- real microphone input
- real transit data
- more languages
- broader speech-native models
- controlled ablations

## Keep Out

- exhaustive metric catalogs;
- raw per-run tables;
- long prompt templates;
- implementation details that do not support the chapter argument.
