# Chapter 3: Evaluation Concepts and Validity Threats - Terminology

Purpose: chapter-local definitions and explanations. Define terms before using them in thesis prose.

## Construct validity

- Definition: The degree to which a metric actually measures the concept it claims to measure.
- Explanation:
  - Route validity has strong construct validity because the network can check it directly.

## Lexical metric

- Definition: A metric based on surface text overlap or token similarity.
- Explanation:
  - Useful for wording comparison, weak for route correctness.

## Semantic metric

- Definition: A metric intended to compare meaning rather than only surface words.
- Explanation:
  - Still not sufficient for executable route validation.

## Metric role

- Definition: The interpretation category assigned to a metric.
- Explanation:
  - Current roles include outcome-confirming, diagnostic phase, associated diagnostic, supplementary, metric quality, and execution/constant.

## Term-Use Rules

- Use one term for one construct; avoid alternating synonyms for key variables.
- Do not use `success`, `route validity`, and `constraint satisfaction` interchangeably.
- Reserve `causal` for controlled comparisons or ablations; otherwise use `associated with` or `indicates`.
- Define abbreviations at first use and prefer full terms in headings.
